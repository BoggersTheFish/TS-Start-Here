# Flagship Route

This is the short public route for skeptical technical visitors.

```text
BoggersTheFish profile README
  -> TS-Start-Here
  -> TS-Reasoner-v0
  -> TensionLM
  -> TS-Codex-OS / TS-Core / BoggersTheCIG
```

## 1. TS-Start-Here

Public ecosystem map, repo taxonomy, and honest status.

Start here when you want to know what exists, what is toy-scope, and what comes next.

## 2. TS-Reasoner-v0

Inspectable toy reasoning telemetry with verifier-backed traces, repair loops, release receipts, a v0.8 externalized small benchmark harness, and a v0.9 proof-chain closure receipt.

Current useful failure: low-tension settlement is not proof completion on proof-chain tasks.

## 3. TensionLM

Sigmoid pairwise tension attention experiments. Treat as narrow model-mechanism research, not a proven replacement for transformer language models.

The public runner should load a Hugging Face checkpoint and produce a raw sample; output quality is not the claim.

## 4. TS-Codex-OS

Local-first project graph, tension ledger, planner, verifier, and release receipt substrate for Codex-driven development.

## 5. TS-Core

Graph/tension runtime kernel for propagation, relaxation, stability, and constraint pressure experiments.

## 6. BoggersTheCIG / CIG Engine

Provenance-aware concept/evidence graph for confidence-weighted claims, contradiction tracking, and inspectable knowledge state.

Use CIG as infrastructure, not as the whole TS system.
